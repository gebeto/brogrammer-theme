# Brogrammer

Originally based on [brogrammer-theme](https://github.com/kenwheeler/brogrammer-theme) by Ken Wheeler.


## Setup

The theme works out of the box in Zed, but to get more granular theming need to enable `semantic_tokens`

```json
// settings.json
...
"semantic_tokens": "combined",
...
```


## Installation

Search for **Brogrammer** in the Zed extension marketplace, or add manually to `~/.config/zed/settings.json`:

```json
{
  "theme": "Brogrammer"
}
```
